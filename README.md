---

## 🚀 About The Project

This portfolio represents my developer identity under the name **Torasoft.exe**.

It was designed with a retro-terminal aesthetic inspired by classic CRT monitors, combining:

- Modern web development principles
- Clean structured layout
- Interactive UI elements
- Smooth animations
- Responsive design

The goal is to present my projects and skills in a unique and memorable way.

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Responsive Design
- Vanilla JavaScript
- Google Fonts (VT323)
- Devicon Icons

No frameworks were used — fully handcrafted.
